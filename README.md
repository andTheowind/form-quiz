# Multi-step-form

Многошаговая форма-опросник на Vue 3 + Vite.

## Установка зависимостей

```bash
npm install
```

## Режим разработки

Запуск локального dev-сервера с горячей перезагрузкой:

```bash
npm run dev
```

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.

## Предпросмотр сборки

Локальный запуск собранной версии:

```bash
npm run preview
```
